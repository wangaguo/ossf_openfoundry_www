Black List 黑名單

Description

管理列入黑名單的使用者. 黑名單的功能並非阻止報名者報名活動, 而是提醒報名者有參加而未到的活動, 希望提高報名者的出席率.


Navigation

使用此功能: Components -> EventList -> 黑名單

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

使用CheckBox選擇想要編輯的VIP CODE

移除黑名單:點選後解除選中的使用者在所有活動的黑名單

幫助: 連結說明頁面.


表單:

姓名 顯示使用者姓名

電子信箱: 顯示使用者信箱

未到次數: 顯示使用者被列了幾次黑名單

Black Note: 顯示使用者在哪個活動被列入黑名單

Black Date: 使用者哪時被列入黑名單(最後一次被列入黑名單的時間)
