List VIP

Description

可在這裡新增VIP CODE或將VIP寄給受邀請的使用者.



Navigation

使用此功能: Components -> EventList -> VIP

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

使用CheckBox選擇想要編輯的VIP CODE

產生VIP CODE:產生n組VIP CODE

給予VIP CODE:產生n組VIP CODE, 在寄給受邀請的對象

刪除VIP CODE:刪除VIP CODE

Help: 連結說明頁面.

表單:

姓名 使用者姓名

電子信箱: 使用者信箱

VIP CODE: VIP碼

狀態: VIP CODE使用狀態,對依據使用者的參加或退出而有所更動

Who Use?: 誰使用這個VIP CODE

select選單

活動名稱:選擇需要使用VIP功能的活動.

受邀姓名:填入受邀者姓名

電子信箱:填入受邀者信箱

數量:需要新增的VIP CODE數量或給予受邀者的VIP CODE數量
