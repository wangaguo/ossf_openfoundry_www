List Events 活動列表

Description

活動管理,列出所有的活動.並新增或編輯.也可以選擇是否要公開活動.



Navigation

使用此功能: Components -> EventList -> 活動

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

使用CheckBox選擇想要編輯的活動

加入管理者: 將自己加入活動聯絡人清單.

封存: 將該活動改為封存狀態, 可在檔案查看.

發佈與停止發佈: 設定活動發佈,若為停止發佈,則在前台無法看見該活動.

新增: 建立新活動

編輯: 點選後編輯選中的活動.

刪除: 點選後將選中的活動刪除.

複製: 點選後將複製選中的活動.

刪除測試資料: 將被選取的活動的參加者資料全部刪除.

GROUP MAIL: 點選後寄出信件給被選中的活動的參加者.

幫助: 連結說明頁面.

表單:

#: 數目.

Checkbox: 選擇活動.可以對活動進行編輯或刪除.

日期: 顯示活動日期.

可能是有其他管理者在編輯該活動編輯, 或是編輯時不正常關閉, 可使用全站回存解除.

活動時間: 顯示活動時間.

標題: 顯示活動標題, 點選後可修改活動設定.

地點: 顯示活動地點, 點選後可編輯地點資訊

鄉鎮市區: 顯示鄉鎮市區.

分類: 顯示活動分類, 點選後可編輯分類資訊

已發佈: 活動是否公開() 或不公開(). 點選後可更改發佈狀態.

創建: 顯示是誰建立的活動以及該活動的講者.

Enrollment: 參加活動的人數.

其他功能

Search: 以關鍵字查詢活動

Drop Down:選擇每頁列出幾個活動.
