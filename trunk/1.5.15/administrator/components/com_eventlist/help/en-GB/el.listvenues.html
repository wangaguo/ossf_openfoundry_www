List Venues 場地列表

Description

編輯活動時, 活動場地的依據, 顯示在前台的活動詳情中. 可新增、編輯、刪除場地資料


Navigation

To access this screen use: Components -> EventList -> 場地

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

使用CheckBox選擇想要編輯的場地.

發佈: 點選後將選中的場地設定為發佈.

停止發佈: 點選後將選中的場地設定為停止發佈.

新增: 點選後新增一個場地.

編輯: 點選後編輯被選中的場地.

移除: 點選後移除被選中的場地.

幫助: Click the Help icon to access a relevant Help Screen.

表單:

#: 數目.

Checkbox: 選擇場地.可以對分類進行編輯或新增.

場地: 顯示場地名稱.

可能是有其他管理者在編輯該活動編輯, 或是編輯時不正常關閉, 可使用全站回存解除.

場地網頁: 該場地的網址.

鄉鎮市區: 顯示該場地位於的鄉鎮市區.

已發佈: 分類是否公開() 或不公開(). 點選後可更改發佈狀態.

事件: 顯示是誰建立此場地.

其它功能

搜尋: 輸入關鍵字後可依據分關鍵字來尋找場地.

Drop Down:可依據設定決定一個頁面顯示多少個分類.
