List Mail 信件管理

Description

管理寄給使用者的信件, 以及使用者的報名訊息. sys欄位顯示每樣功能所使用的信件. 可在"設定"功能中 修改所使用的信.


Navigation

使用此功能: Components -> EventList -> 信件管理

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

使用CheckBox選擇想要編輯的信件

新增: 點選後新增信件.

編輯: 點選後將編輯選中的信件.

刪除: 點選後將選中的信件刪除..

幫助: Click the Help icon to access a relevant Help Screen.

表單:

#: 數目.

Checkbox: 選擇想編輯的信件.

信件名稱: 顯示信件名稱.
