Archive 檔案

Description

管理所有的封存活動列表, 可以選擇解除封存或刪除資料.


Navigation

To access this screen use: Components -> EventList -> Archive

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

使用CheckBox選擇想要編輯的活動.

Unarchive: 點選後將選中的活動解除封存.

移除: 點選後將選中的活動刪除.

Help: Click the Help icon to access a relevant Help Screen.


表單

#: 數目.

Checkbox: 使用CheckBox選擇想要編輯的封存活動.

活動日期: 顯示活動日期.

標題: 顯示活動標題.

活動時間: 顯示活動時間.

地點: 顯示活動地點.

分類: 顯示活動分類.

鄉鎮市區: 顯示鄉鎮市區.

創建: 顯示作者.

其它功能

搜尋: 輸入關鍵字後可依據分關鍵字來尋找場地.

Drop Down:可依據設定決定一個頁面顯示多少個分類.
