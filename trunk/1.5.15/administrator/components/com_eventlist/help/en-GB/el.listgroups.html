List Groups 群組列表

Description

管理群組列表, 可新增刪除、編輯、群組

Navigation

To access this screen use: Components -> EventList -> 群組

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

使用CheckBox選擇想要編輯的群組.

新增: 點選之後可新增新的群組.

編輯: 點選後將編輯選中的群組.

移除: 點選後將選中的群組刪除.

幫助: Click the Help icon to access a relevant Help Screen.


表單:

#: 數目.

Checkbox: 選擇群組.可以對群組進行編輯或刪除.

群組名稱:顯示群組名稱

可能是有其他管理者在編輯該群組編輯, 或是編輯時不正常關閉, 可使用全站回存解除.

細節: 顯示群組細節.

其他

Eventlist中有預設三個群組, 用於系統訊息通知.

報名通知群組: 當有新成員報名活動時, 系統會發系統信給該群組的成員. 當活動額滿停止報名時 系統也會發信告知該群組的成員.

活動聯絡人: 顯示於前台的活動聯絡人頁面.

講者:活動講者列表. 於編輯活動時使用.
