Categories 分類列表

Description

管理活動分類, 可編輯、新增、刪除分類, 在編輯活動時的分類列表. 也可作為篩選活動的依據.


Navigation

To access this screen use: Components -> EventList -> 類別

User Access Level

Super Administrator Administrator Manager

Toolbar Icons

使用CheckBox選擇想要編輯的分類.

發佈與停止發佈: 設定類別發佈,若為停止發佈,則在前台無法看見該類別. 編輯活動時也無法選擇該分類.

編輯: 點選後可編輯選中的分類.

新增: 點選後可新增一個分類.

刪除: 點選後可刪除選中的分類.

幫助: Click the Help icon to access a relevant Help Screen.

表單:

#: 數目.

Checkbox: 選擇分類.可以對分類進行編輯或刪除.

分類: 顯示分類名稱.

可能是有其他管理者在編輯該活動編輯, 或是編輯時不正常關閉, 可使用全站回存解除.

別名: 顯示分類別名.

社群: 顯示分類屬於哪種群組, 一個分類只能擁有一個群組.

活動: 顯示該分類有多少活動.

已發佈: 分類是否公開() 或不公開(). 點選後可更改發佈狀態.

存取: 顯示目前該分類開放給何種使用者, 有 註冊會員、特定的、公開的三種選項.

其它功能

搜尋: 輸入關鍵字後可依據分關鍵字來尋找分類.

Drop Down:可依據設定決定一個頁面顯示多少個分類.
