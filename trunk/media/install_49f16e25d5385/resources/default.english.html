Joomla! 1.0 to 1.5 Migrator

This component handles migration of your 1.0 data to a 1.5 site.

Migrating your data is quite simple:

  1. Install this component
  2. Install any third party ETL plugins and table creation files
  3. Run the migration creation system
  4. Download the resultant SQL file
  5. Upload the SQL file as a part of your 1.5 installation.
  6. If you have no third party components to migrate your job here is done!
  7. Manually install special migration install files on your 1.5 site (installer packages with SQL "CREATE TABLE IF NOT EXISTS" statements)

Note: We cannot ensure that third party extensions will work properly or as they did. Additionally 1.5 changes a lot of functionality within Joomla! and some items will be moved. This system does an almost complete database transfer to your new 1.5 site but some settings cannot be transferred and need to be manually re-entered.
